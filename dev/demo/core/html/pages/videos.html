Video Player
The video player components is a completely custom HTML5 video player with CSS controls and completely customizable functionality. Use new VideoPlayer(options) to create a new instance of a video player, and pass an object as the options parameter. The target property of the parameter object should take in the HTML Node of the containing element of the video player. It should not be a <video> element itself. Either a string of the URL path to a single video should be passed to the video property or an array of strings of URL paths to videos can be passed to the playlist property.
The following properties are all optional:
autoplay Boolean: Whether the video or first video should automatically start playing
controls Boolean: Whether there should be controls at the bottom of the player
loop Boolean: Whether the video or playlist should automatically restart upon completion
muted Boolean: Whether the video or playlist should start playing with the sound off
color String: CSS value for a color to use for the controls of the video player
