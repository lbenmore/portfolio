The BNMR Library began as a small utility library written to automate frequently-used functions in environments that were not jQuery-friendly (or other "lightweight" libraries).

When creating ads, there is a convention for how many kilobytes a banner should load. It is a very small number when considering that high-resolution images are now a must. For this reason, jQuery and other libraries that claim to be "lightweight" could not be used. For many new developers, this took away the familiar methodologies they had for creating simple animations.
