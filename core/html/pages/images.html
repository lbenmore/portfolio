These components can be easily implemented into any project utilizing the Bnmr Library. They each have their own encapsulated styles as well as a library file.
Carousel
To use the image carousel component, create a new instance using new Carousel(options) and pass the desired parameters via the options object. The expected properties of the options object are target, images, arrows, and pagination. The target and images parameters are both required. An HTML node should be passed to target and will be the containing element for the carousel. An array of URL paths as strings should be passed to the images property. The other two properties are optional. If provided, arrows is a boolean of whether right and left arrows should be present on the carousel. If the pagination boolean determine if there is navigable image indicators on the carousel. Both optional parameters are true by default.
Lazy Loader
The lazy loading components is also a factory function that allows you to create new instances using new LazyLoader(options) where options is an object. The options parameter object expects two properties: target and items. Pass an HTML node to the target property that will contain the content. Currently, the items property expects an array of image paths. This will be updated in the future to take in differnt types of content. Both parameters are required.
